⚙️ Initial Setup (One-Time)

The first time you use the predictor, you need to discover what event names BloxLuck uses.

1

Load the bookmarklet in DISCOVERY mode

Open BloxLuck, tap the bookmarklet. The panel will show "DISCOVERY MODE".

2

Play 3-5 coinflips

Each coinflip will log events to the panel. Watch for:

  • BET EVENT — the event sent when you join a game (e.g., coinflip:join)
  • RESULT EVENT — the event received with the outcome (e.g., coinflip:result)
  • SIDE FIELD — how the side is named (e.g., side, choice, team)
3

Save your config

Go to the Config Panel and save the event names. They'll be stored in your browser for next time.
